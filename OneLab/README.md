# OneLab Project

This folder contains the OneLab application and its resources. It includes:

- **app/**: Main application code, models, routes, services, static files, and templates.
- **instance/**: Application database.
- **requirements.txt**: Python dependencies for the project.
- **run.py, seed_data.py, streamlit_app.py**: Scripts for running and seeding the application, and a Streamlit interface.

Please refer to the subfolders for more details. Ensure no sensitive data is present before publishing.
